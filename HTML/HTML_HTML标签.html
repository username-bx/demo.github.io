    • HTML标签

    • HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
    • HTML 标签是由尖括号包围的关键词，比如 <html></html>
    • HTML 标签通常是成对出现的，比如 <b> 和 </b>
    • 标签对中的第一个标签是开始标签，第二个标签是结束标签

    • 根标签

    • <html> </html>
    • 所有的内容都写在根标签里面

    • 结构化标签

    • 结构化标签有两个，<head></head> ,<body></body>
    • head里面存放的是给浏览器看的
    • body里面存放的是给用户看的

    • 单标签<meta>

    • <meta charset='UTF-8'> 表示给浏览器一本字典，这样浏览器就能识别各国语言

    • <html lang='en'>

    • <html lang='en'> 告诉搜索引擎爬虫，我们的网站是关于什么内容的，更方便它 来抓取我们的网站。

    • 特殊编码

    • 三个特殊编码：& nbsp; & lt; & gt;
    • & nbsp; -- 空格
    • & lt; -- 左尖角号
    • & gt; -- 右尖角号

    • 单标签

    • 一对标签：致力于把包裹的内容作用成它想作用成的样子
    • 单标签：自己就是一个功能，不需要包裹内容
    • <br> -- 回车
    • <hr> -- 一条线

    • 有序列表

    • <ol>有三个属性，type，reversed，start
    • type='1' -- 按照1,2，3排序
    • type='i' -- 按照罗马数字排序
    • type='a' -- 按照a, b, c排序
    • type='A' -- 按照A, B, C排序
    • reversed='reversed' -- 反向排序
    • strart='3' -- 从序号的第三位开始排序

    • 无序列表

    • <ol>有三个属性，type，reversed，start
    • type='disc' -- 实心圆排序
    • type='circle' -- 空心圈排序
    • type='square' -- 实心方块排序
    • reversed='reversed' -- 反向排序
    • strart='3' -- 从序号的第三位开始排序

    • 图片

    • src='' -- 图片路径
    • alt='' -- 图片占位符
    • title='' -- 图片提示符
